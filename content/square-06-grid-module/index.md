페이지를 수학적으로 분할하여 텍스트·이미지·여백의 비례를 체계화. 스위스 국제 타이포그래피 양식의 핵심. CSS Grid의 직계 조상.

## 맥락

20세기 중반 스위스의 취리히·바젤을 중심으로 형성된 국제 타이포그래피 양식(International Typographic Style)에서 격자 시스템이 규범으로 정착했다. 얀 치홀트(Jan Tschichold)의 『신 타이포그래피』(1928), 막스 빌(Max Bill)·에밀 루더(Emil Ruder)의 실천을 거쳐, 요제프 뮐러-브로크만(Josef Müller-Brockmann)이 『Grid Systems in Graphic Design』(1961/1981)에서 집대성했다. 배경에는 합리주의·기능주의 건축(바우하우스, 데스틸), 중세 필사본의 페이지 구성, 그리고 구텐베르크 이래의 인쇄 조판 규범이 겹쳐 있다. "개인 취향이 아니라 체계가 결정한다"는 모더니즘의 윤리가 판면에 구현된 형태다.

## 쓰임

에디토리얼 디자인, 브랜딩 시스템, 포스터, 웹·앱 UI 설계에 예외 없이 관여한다. 12컬럼 웹 그리드, 8pt 스페이싱 시스템, 피그마의 오토 레이아웃, CSS Grid와 Flexbox가 모두 이 전통의 연장이다. 디자인 시스템의 레이아웃 토큰(컬럼·거터·마진·베이스라인)은 곧 격자의 디지털 번역이다. 단일 포스터에서 수백 페이지의 시리즈, 반응형 웹사이트까지 일관성을 유지하는 방법으로 격자가 먼저 호출된다.

## 편집점

집단적 창조를 위한 편집의 단위 — 그리드는 개인 취향을 제도화한 문법이다. 모듈 단위로 나누면 혼자 작업하든 팀으로 작업하든 결과가 들쭉날쭉해지지 않는다. 디자이너가 바뀌어도, 페이지가 늘어나도, 서비스가 여러 채널로 확장되어도 — 모듈이라는 최소 단위 덕분에 전체의 일관성이 유지된다. 그리드는 창의성의 반대가 아니라, 협업의 기반이다. 좋은 그리드는 제약처럼 보이지만 실은 확장을 가능하게 하는 구조다.
